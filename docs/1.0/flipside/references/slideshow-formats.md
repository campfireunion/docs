# Slideshow formats

The slideshow supports the following formats:

Photo:

* JPG
* PNG

Video:

* MP4
* MKV
* WEBM
* RTMP (Live stream)
* RTSP (Live stream)
* Dropbox (ex: `https://www.dropbox.com/s/path/to/my-image.png?dl=0`)
* Google Drive (ex: `https://drive.google.com/file/d/file-id/view?usp=sharing`)

Audio codecs:

* MP3
* AAC
